# 跨國法令與稅務風險監測系統（Tax Monitor API）

本專案為一個以 FastAPI
建立的後端系統，主要目的是協助使用者快速理解跨國法令內容，並進一步分析可能的稅務風險。

系統提供文件上傳、關鍵字抽取、摘要分析與風險判斷等功能，讓使用者在面對大量法規資料時，可以更有效率地掌握重點。

------------------------------------------------------------------------

## 專案目標

1.  自動整理跨國法規內容
2.  協助辨識潛在稅務風險
3.  降低人工閱讀與理解成本
4.  提供後續 AI 擴充的基礎架構

------------------------------------------------------------------------

## 系統設計概念

### 上下文優化（Context Optimization）

系統會先抽取關鍵字與重要句子，再進行分析，以降低雜訊並提升準確度。

### 逆向驗證機制（Reverse Checking）

分析完成後，會回頭比對原文，確認結論是否有依據，降低錯誤風險。

### 分析流程

-   translate_first：先翻譯再分析
-   analyze_first：先分析再翻譯

------------------------------------------------------------------------

## 系統架構

FastAPI Backend

-   Document Layer：文件解析與關鍵字抽取\
-   Analysis Layer：摘要、風險判斷、翻譯\
-   Storage Layer：SQLite

## 使用流程

1.  上傳文件（POST /api/document/upload）
2.  查看列表（GET /api/document/list）
3.  執行分析（POST /api/analysis/run）
